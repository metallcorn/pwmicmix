Remove everything?

This deletes all channels and their virtual microphones. Apps currently using them will lose the input. This can't be undone.

Quit AudioMixer?

The app will close. Your virtual microphones keep running, so apps using them won't lose their input — relaunch AudioMixer to control them again. To remove the mics entirely, use ⏹ Stop or 🗑 Clear first.

Restart AudioMixer?

The server restarts in place. Your virtual microphones keep running and the window reconnects in a moment.

Connect a phone

Scan with a phone on the same network — it opens the mixer and signs in with the PIN automatically. Pick the right network if you have several (hotspot / Wi-Fi / cable).
